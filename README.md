# Welcome to the MVP of the Entra Conditional Access Policy Optimizer

**Key Features of the MVP alpha release:**
- Accepts a prompt about security requirements in plain language
- AI analyses the prompt and recommends CA policies
- Policies are presented in a matrix
- Tests are performed to evaluate hardening

There are many more features to add! This was built in [Google's Gemini AI Studio](https://aistudio.google.com). Feedback welcome: <a href="mailto:hello@warrenspier.com">hello@warrenspier.com</a>.

# Run and deploy your AI Studio app

This contains everything you need to run your app locally.

View your app in AI Studio: https://ai.studio/apps/drive/1RZIp2tNaOaUmZvvzQRVPrwh3R1bJYr1z

## Run Locally

**Prerequisites:**  Node.js

1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Run the app:
   `npm run dev`
